# FutStat

MVP trackera meczow FC/FUT Champions z szybkim dodawaniem wynikow i prostym dashboardem.

## Co jest teraz

- szybkie dodanie meczu
- lokalny zapis w `localStorage`
- rekord, streak, najlepsza godzina gry, najtrudniejszy matchup
- prosta rekomendacja sesji na podstawie ostatnich meczow

## Start

```bash
npm install
npm run dev
```

Domyslnie aplikacja uruchamia sie na `http://localhost:4026`.

## Kolejny krok

- statystyki zawodnikow
- import ze screena po meczu
- konto usera i sync danych
